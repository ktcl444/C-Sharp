Tools and Templates

The Tools and Templates folder structure, file descriptions, and usage is described in Chapter 5, "Implementing the Solution" in each paper. Tools and Templates are typically provided for each solution scenario.

Warning  These samples are provided as examples only. Be sure to review, customize, and test the provided tools before using them in any non-test environment.

Note  In version 1.4, the default install path for the Tools and Templates is %UserProfile%\My Documents\Identity and Access Management Tools and Templates.
